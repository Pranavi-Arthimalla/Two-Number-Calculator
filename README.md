# Two-Number-Calculator
🎀 A simple two‑number web calculator built with HTML, CSS, and JavaScript. Focused on styling and basic arithmetic operations.

💡 Purpose
This project was created to practice CSS styling and make a basic calculator look fun and cute.


## ✨ Features
- Perform basic arithmetic operations:
  - ➕ Addition
  - ➖ Subtraction
  - ✖️ Multiplication
  - ➗ Division
  - % Modulus
  - ^ Exponentiation
- Cute styled interface with hover effects
- Error handling for division/modulus by zero
